Surgimento da internet

(voltar)

Tudo começou nos anos 1960, durante a Guerra Fria, quando os Estados Unidos sentiram a necessidade de criar uma rede de comunicação que pudesse resistir a ataques nucleares. Esse projeto, inicialmente financiado pelo Departamento de Defesa dos EUA, deu origem ao ARPANET (Advanced Research Projects Agency Network).

Anos 1970: Nos anos seguintes, a ARPANET cresceu e conectou várias universidades e centros de pesquisa. Foi nessa época que surgiram os protocolos de comunicação, como o TCP/IP, desenvolvidos por Vint Cerf e Bob Kahn, que ainda são a base da internet moderna.

Anos 1980: A ARPANET começou a se expandir para além dos círculos acadêmicos e militares. No mesmo período, Tim Berners-Lee, um cientista britânico, inventou a World Wide Web (WWW) enquanto trabalhava no CERN. Ele criou o primeiro navegador e editor web, além de desenvolver os padrões HTML, HTTP e URLs.

Anos 1990: Com a introdução dos navegadores web amigáveis, como o Netscape, a internet começou a ganhar popularidade entre o público em geral. Empresas e indivíduos começaram a criar sites, e o número de usuários cresceu exponencialmente.

Anos 2000 em diante: A internet evoluiu rapidamente com a chegada da banda larga, redes sociais, e-commerce e plataformas de streaming. Hoje, a internet é uma parte integral das nossas vidas, conectando bilhões de pessoas e dispositivos em todo o mundo.

Como funciona o html

(voltar)

Para criar um arquivo HTML, você começa com <!DOCTYPE html>, que informa ao navegador que está usando HTML5. O elemento raiz <html> envolve todo o conteúdo da página, com lang="pt-BR" especificando o idioma português do Brasil.

Dentro do <html>, há duas seções principais: <head> e <body>. O <head> contém metadados como <meta charset="UTF-8">, que define a codificação de caracteres, e <meta name="viewport" content="width=device-width, initial-scale=1.0">, que ajusta a página à largura do dispositivo para ser responsiva. A tag <title> define o título da página, que aparece na aba do navegador.

O <body> contém todo o conteúdo visível da página, como títulos, parágrafos, imagens e links. Ele é onde todos os elementos que os visitantes verão estão localizados.

Resumindo, a estrutura básica de um arquivo HTML começa com a declaração do tipo de documento, seguida pelo elemento raiz <html> que contém <head> para metadados e configurações, e <body> para o conteúdo visível.

Tags HTML são blocos de construção básicos de uma página web, funcionando como marcadores que instruem o navegador sobre como exibir o conteúdo. As tags HTML geralmente consistem em uma tag de abertura e uma tag de fechamento. A tag de abertura marca o início de um elemento, enquanto a tag de fechamento marca o fim desse elemento. Por exemplo, para criar um parágrafo, usamos a tag <p> para abrir e

para fechar.

O que são Tag's

(voltar)

As tags HTML podem ter atributos que fornecem informações adicionais sobre o elemento. Esses atributos são incluídos dentro da tag de abertura. Por exemplo, a tag <img> usa o atributo src para definir o caminho da imagem e alt para fornecer uma descrição alternativa.

Em resumo, as tags HTML são essenciais para estruturar e exibir o conteúdo de uma página web, indicando ao navegador como cada parte da página deve ser apresentada, desde texto e imagens até links e listas
